S&P 500 모의투자 및 AI 추천 플랫폼 "MARS"

2025.03 - 2025.07 [부트캠프 팀 프로젝트] [팀장]

S&P 500 실시간 데이터 기반 모의투자 웹 서비스. 가상 매매 시뮬레이션과 LLM 기반 종목 추천 기능을 포함하며, 본인은 팀장으로서 백엔드, 데이터 파이프라인, AI 추천 엔진, 배포 인프라를 담당.

NestJS Next.js Prisma MySQL Python LangChain Docker Nginx AWS EC2 GitHub Actions

Challenge & Action: NestJS와 Prisma 기반으로 모의투자의 핵심 비즈니스 로직을 구현. 수수료 계산이 포함된 매수/매도 주문 처리, 실시간 수익률 산출, 그리고 거래 API에 낙관적 락 패턴을 적용해 "클라이언트가 본 시점의 가격"과 "서버의 현재 가격"이 다를 때 거래를 거부하도록 설계.(결과적으로 낙관적 락 패턴을 적용) 프론트엔드 팀원들과의 연동을 위해 백엔드 API 전체 명세를 Markdown으로 직접 작성해 공유했고, 그 덕에 연동 과정에서 질문이 거의 발생하지 않음. CI/CD는 GitHub Actions로 구축해 특정 브랜치 푸시 시 AWS EC2에 자동 배포.

1년차 팀장으로서 예상치 못한 상황에서의 역할 재배치와 우선순위 조정을 경험했습니다. 원래 별도 담당자가 있던 AI 추천 엔진을 포함해 데이터 수집, 백엔드, 인프라까지 결국 혼자 담당하게 됐고, 그 과정에서 불완전한 시스템이라도 요구사항 기한 안에 동작하는 상태로 만드는 판단력과, 팀 협업을 위해 무엇을 우선하고 무엇을 감수할지 결정하는 감각을 얻었습니다. 이 프로젝트의 실제 구조, 기술적 판단, 그리고 팀 리드로서 부딪힌 현실의 전반적 회고는 회고 문서에 정리되어 있습니다.

금융권 코어 뱅킹 시스템 유지보수 및 개선 작업

2023.11 - 2024.11 [아네스트코리아] [실무 경력]

10년간 운영되어 온 인터넷 은행 코어 뱅킹 솔루션의 유지보수를 담당. 신상품 로직 추가와 요구사항 변경 대응을 수행하면서, SVN→Git 이관과 프론트엔드 리팩터링 같은 개선 작업도 병행.

JSP Oracle SVN/Git GitLab JavaScript 전문통신 COBOL

Challenge & Action: 문서가 부족한 환경에서 코드로 로직을 역추적하며 JSP 기반 코어 뱅킹 시스템의 유지보수를 담당. 신상품 로직 추가, 요구사항 변경 대응, 버그 수정 같은 일상적 유지보수 업무와 함께, 10년간 폴더 단위 버전업 방식으로 비대해진 SVN 리포지토리의 전체 이력을 Git으로 이관. 팀 대부분이 Git에 익숙하지 않은 상태였기 때문에, 선임이 제시한 트렁크 기반 전략을 직접 공부한 뒤 팀원들에게 설명하기 위한 문서를 작성하고 회의에서 발표. 또한 IE 전용 비표준 스크립트를 모던 JS로 리팩터링하는 작업을 수행하며, Observer 패턴 기반 구조의 개별 모듈을 구현하고 브라우저 호환성 문제를 해결.

문서보다 코드의 영향력이 큰 환경에서 1년간 일하며, 실제 운영되고 있는 시스템에 변경을 가하는 일의 무게를 체감했습니다. 오래 유지된 시스템의 결정들에는 당시의 이유가 있었다는 것, 그리고 환경이 바뀌면 그 이유가 다시 검토되어야 한다는 것을 함께 배웠습니다. 이 감각은 이후 신규 프로젝트에서도 "지금 내리는 결정이 3개월 뒤에 어떻게 드러날까"를 고려하는 기본기로 작용하고 있습니다.
